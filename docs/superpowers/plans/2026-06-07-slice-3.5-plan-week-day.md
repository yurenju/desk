# Slice 3.5 — Plan 週導覽 + Day 欄可規劃 實作計畫

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 讓 Plan 三欄（Month / Week / Day）繞單一「焦點日」耦合、焦點日進 URL，且 Plan 的 Day 欄完全可互動（設每日三件事、勾完成、編輯、刪除、新增）。

**Tech Stack:** React 18 + TypeScript、TanStack Router（file-based + `tsr` 產生 `routeTree.gen.ts`）、Zustand、Vitest + Testing Library、Playwright e2e。

在 `src/lib/date.ts` 結尾加（用本地時間、避免 UTC 位移；`addMonths` 對日做 clamp）：

```ts
/** Shift an ISO date (YYYY-MM-DD) by n days. */
export function addDays(date: string, n: number): string {
  const d = new Date(date + "T00:00:00");
  d.setDate(d.getDate() + n);
  return todayISO(d);
}

/**
 * Shift an ISO date by n months, clamping the day to the target month's last
 * day (e.g. addMonths("2026-01-31", 1) === "2026-02-28").
 */
export function addMonths(date: string, n: number): string {
  const [y, m, day] = date.split("-").map(Number);
  const targetIdx = y * 12 + (m - 1) + n;
  const ny = Math.floor(targetIdx / 12);
  const nm = targetIdx % 12; // 0-based month
  const lastDay = new Date(ny, nm + 1, 0).getDate();
  const clampedDay = Math.min(day, lastDay);
  return `${ny}-${String(nm + 1).padStart(2, "0")}-${String(clampedDay).padStart(2, "0")}`;
}
```
